# SBMN Interviewer Experimental Dataset

## Overview

This repository contains the experimental artifacts used to evaluate the **SBMN Interviewer**, an LLM-based approach for business process elicitation through Structured Business Model Notation (SBMN).

The dataset is organized into independent scenarios, each containing all artifacts required to reproduce, validate, and analyze a complete experiment, from the original process description to the final evaluation results.

The repository was designed to support **reproducibility**, **artifact reuse**, **benchmarking**, and **independent validation** of the reported results, following FAIR principles.

## Scenario Components

### Description

Contains the textual description of the business process used as the source for the interview.

| File                      | Description                                           |
| ------------------------- | ----------------------------------------------------- |
| `process_description.txt` | Natural language description of the business process. |

---

### Reference

Contains the ground-truth artifacts used during evaluation.

| File                             | Description                                                            |
| -------------------------------- | ---------------------------------------------------------------------- |
| `reference_model.bpmn`           | Original BPMN process model.                                           |
| `expected_sbmn.json`             | Expected SBMN model derived from the reference process.                |
| `scenario_characterization.json` | Structural characteristics and complexity information of the scenario. |

---

### Interview

Contains the artifacts produced during the elicitation phase.

| File                       | Description                                                      |
| -------------------------- | ---------------------------------------------------------------- |
| `interview_transcript.txt` | Complete interview transcript generated by the SBMN Interviewer. |
| `obtained_sbmn.json`       | SBMN model obtained from the interview.                          |
| `interview_metadata.json`  | Metadata describing the interview execution.                     |

---

### Solutions

Contains BPMN models generated from the elicited SBMN model.

| File              | Description                                            |
| ----------------- | ------------------------------------------------------ |
| `solution_*.bpmn` | BPMN solutions generated from the obtained SBMN model. |

---

### Evaluation

Contains the results of the experimental evaluation.

| File                      | Description                                           |
| ------------------------- | ----------------------------------------------------- |
| `evaluation_metrics.csv`  | Quantitative evaluation metrics.                      |
| `evaluation_summary.json` | Consolidated evaluation results.                      |
| `confusion_analysis.csv`  | Analysis of missing, correct, and spurious relations. |

---

### Metadata

Contains additional information required for experiment management and traceability.

| File                     | Description                          |
| ------------------------ | ------------------------------------ |
| `scenario_metadata.json` | General metadata about the scenario. |

## FAIR Compliance

The dataset follows FAIR principles by providing:

* **Findable** artifacts through a standardized directory structure.
* **Accessible** machine-readable files (TXT, JSON, BPMN, CSV).
* **Interoperable** representations based on open formats.
* **Reusable** artifacts with complete metadata and traceable provenance.
